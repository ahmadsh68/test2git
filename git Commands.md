#Git Commands

## SetUP
- git init
- git remote add <repo>
- git add <files>
- git commit -m "first message commit"
- git push origin master

## Git Branches
### Create new Branches
- git branch <branch Name> 
- git checkout <branch Name>
- git checkout -b <branch Nmae> = git branch <branchname > then git checkout <branchname>
